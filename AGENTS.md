
## Recap
- This the Media Tracker application. In `./front-end` we have the web application UI and in `./back-end` we have the back-end APIs. Both front-end and back-end have their own self-consistent projects.

## Core Constraints
- Whenever asked to work on the front-end, you should work in the `./front-end` directory and follow `./front-end/AGENTS.md`.
- Whenever asked to work on the back-end, you should work in the `./back-end` directory and follow `./back-end/AGENTS.md`.
